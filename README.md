📊 Sales Data Analysis Project

🚀 Overview

This project focuses on analyzing sales data using SQL (MySQL) and visualizing insights through Power BI.
The dataset was cleaned, transformed, and analyzed to extract meaningful business insights.

📌 Detailed explanation including SQL queries, outputs, and final dashboard screenshots is provided separately.

🛠 Tech Stack

MySQL
SQL (Subqueries, Aggregations)
Excel / Power Query
Power BI

📂 Key Highlights

Data cleaning and preprocessing
Advanced SQL analysis using subqueries
KPI calculation (Revenue, Profit, Orders, AOV)
Multi-dimensional analysis (Country, Category, Sales, Device)
Dashboard creation using Power BI

🔍 Key Business Problems Solved

Identified top 5 customers whose revenue is above average
Found the country with the highest total profit
Analyzed sales representatives outperforming their manager’s average
Determined top 3 product categories by profit
Extracted top 5 orders with profit above overall average

📊 Output

Business insights derived using SQL
Interactive Power BI dashboard
